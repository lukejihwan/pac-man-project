# pac-man-project

### 프로젝트 설명
팩맨이 유령을 피해 먹이를 모두 먹으면 승리하는 게임

### 프로젝트 주요 기능
1. 유령, 벽, 팩맨 충돌기능 구현(코드 추가)
2. 유령 자동 움직임 구현
3. 팩맨 먹이 획득시 점수 증가 구현
4. javascript 클래스 분리 구현
